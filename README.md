# Vue环境搭建文档

本工程以vue-cli 3.0 为基础构建项目

1.更新vue-cli
```
$ npm install -g vue@cli
```
2.初始化项目
```
$ vue create <you project>
```

3.最大不同不是以前的选模板了,而是选presets(预设),或者自己手动设置.(这里选择默认的default (babel,eslint))
